# 前任检测仪 (ExScan)

一个基于 React + TypeScript + Tailwind CSS 的娱乐性前任类型检测应用。

## 🎯 项目简介

前任检测仪是一个轻松有趣的Web应用，用户可以输入前任的相关信息，系统会随机生成一个幽默的前任类型分析报告，包含类型描述和"毒鸡汤"建议。

## ✨ 功能特性

- 🎨 **现代化UI设计**：粉紫渐变主题，响应式布局
- 🎲 **随机检测系统**：12种不同的前任类型
- ⏳ **加载动画**：优雅的检测过程展示
- 🔄 **重新测试**：支持多次检测
- 📱 **响应式设计**：完美适配各种设备

## 🛠️ 技术栈

- **前端框架**：React 18
- **类型系统**：TypeScript
- **样式框架**：Tailwind CSS 3.4.17
- **构建工具**：Vite
- **包管理器**：pnpm

## 🚀 快速开始

### 环境要求

- Node.js >= 16
- pnpm >= 7

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本

```bash
pnpm build
```

### 预览生产版本

```bash
pnpm preview
```
